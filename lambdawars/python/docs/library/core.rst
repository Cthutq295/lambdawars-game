:mod:`core` --- Core Game Package
===================================================

.. module:: core
   :synopsis: Core Game Package

The *core* gamepackage defines most of the base classes for Half-life 2: Wars. Most of
the time when creating a new units, ability or anything else you will use this gamepackage.
